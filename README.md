# Lyrics-by-JL

Bem vindos ao meu site pessoal. Esse site visa exibir para o público um pouco sobre a forma como o "eu lírico" criado por mim observa o mundo e entende seus sentimentos.

# Meu Site Pessoal

Esse site foi criado para compartilhar:

**Home:** Um pouco sobre sentimentos expressado atraves da música
**Composições:** As letras de música de minha autoria
**Sobre mim:** Um pouco sobre mim

O site foi desenvolvido utilizando as seguintes tecnologias:
- **HTML:** Estrutura básica do site.
- **CSS:** Estilização e layout.
- **JavaScript:** Interatividade e funcionalidades adicionais.
- **React:** Compenentização, organização e reaproveitamento do código.
